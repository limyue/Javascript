在sublime中安装js,css检错插件
http://www.cnblogs.com/uestcbutcher/p/6971227.html
步骤：
（1） 安装SublimeLinter
Package Control: Install Package
输入SublimeLinter
（2）安装SublimeLinter-jshint
Package Control: Install Package
SublimeLinter-jshint
（3）nodejs下安装 jshint
npm install -g jshint
jshint -v
（4）安装css语法检验SublimeLinter-csslint
Package Control: Install Package
SublimeLinter-csslint
（5）nodejs下安装csslint
npm install -g csslint   
csslint --version
